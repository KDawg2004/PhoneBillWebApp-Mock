Phone Bill App\n\n
    This app allows you to manage phone bills.\n
    Add Call: Enter caller, callee, start and end times
    in M/d/yyyy h:mm a format (3/11/2026 6:30 PM).
    Search Calls: Enter a customer name to view all calls.
    (Optionally) enter a date range to filter results.
    Data is saved automatically to internal storage.
